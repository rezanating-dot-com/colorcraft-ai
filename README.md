# ColorCraft AI

Generate beautiful, printable coloring book pages from text descriptions using AI image generation.

![Tailwind CSS](https://img.shields.io/badge/Tailwind%20CSS-4-38bdf8)

## Features

- **AI Image Generation** — Describe any scene and get a professional coloring book page generated by Google Gemini
- **Print-Ready Output** — Clean black-and-white line art optimized for printing
- **Auto-Save Gallery** — Every page is automatically saved to your browser's local storage
- **Download & Print** — Export as PNG or print directly from the browser
- **Favorites** — Mark your best creations for easy access
- **Privacy-First** — No account needed. All data stays on your device

## Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **UI:** [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **AI:** [Google Gemini 2.5 Flash Image](https://ai.google.dev/gemini-api/docs/image-generation)
- **Storage:** IndexedDB via [idb-keyval](https://github.com/nicedoc/idb-keyval)
- **Icons:** [Lucide React](https://lucide.dev/)

## Getting Started

### Prerequisites

- Node.js 18+
- A [Google Gemini API key](https://aistudio.google.com/apikey)

### Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/kshekan/colorcraft-ai.git
   cd colorcraft-ai
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Create a `.env.local` file with your API key:

   ```
   GEMINI_API_KEY=your_api_key_here
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000)

## How It Works

1. **Describe** your coloring page in the text input
2. **Generate** — the app sends your description to Google Gemini, which creates a black-and-white line art image
3. **Download or Print** — save the PNG or print it directly
4. **Gallery** — all generated pages are saved automatically and can be favorited, downloaded, or deleted

## Deploy on Vercel

1. Push this repo to GitHub
2. Import it on [Vercel](https://vercel.com/new)
3. Add `GEMINI_API_KEY` to your environment variables
4. Deploy
